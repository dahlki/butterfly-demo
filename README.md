# butterfly-demo

## what to do
```shell
 $ git clone https://github.com/dahlki/butterfly-demo.git
 
 $ cd butterfly-demo
 
 $ npm install or yarn
 
 $ npm start
 
   or to build and start server separately:
   
 $ npm run build-cient  
 
 $ npm run start-server
 
   open localhost:3000
   or to directly pick a mood from 1-5
   open localhost:3000/v/your-mood-level-here
 ```
